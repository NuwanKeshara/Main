git basic commands

git add <file name>  or git add -a  -- adding new file for the first time (-a for all)
git commit -m "<description>" <file name>  or git commit -a -m "<description>" <file name> -- after modification of existing file or afte git add command(-m for message)
git pull  --before every push
git push -u origin master   -- upload to the github 
git clone <url>   -- download from github 


if git push rejected :
*git pull
*git add -A
*git push
